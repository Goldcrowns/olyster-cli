# olyster-cli
# ⚡ Olyster CLI and Agent

🚀 **An open-source, terminal-based AI agent and multi-model powerhouse.**
Control your terminal, analyze codebases, ask questions, or change models instantly—all without leaving your command line. Powered by Gemini and Groq.

> *Why constantly switch between a browser and a heavy IDE code assistant when you have an incredibly fast and lightweight AI agent and a full-scale web platform living in your terminal?* 🛠️🌐

---

## ✨ Features

* **📺 Interactive TUI:** A beautiful terminal UI built with `@neuledge/blessed`. Features a custom navigation/input bar at the bottom to keep your screen clean and free from token scrolling clutter. * **🧠 Multiple Model Cores:** Instantly switch between high-level models like **Llama 3.1**, **Llama 3.3 Pro**, and **Gemini Flash Lite** with quick commands.

* **🛠️ Native Tool Call (In Active Development):** Olyster doesn't just talk, it acts. It's designed to securely execute terminal commands, read/write files, and browse native codebases according to your requests.

* **🌐 Ecosystem Integration:** Seamlessly paired with the [Olyster Web Platform](https://olyster.vercel.app).
